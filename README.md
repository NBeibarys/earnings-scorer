# Earnings Call Investment Scorer

**NYU · Text as Data · Spring 2026**

Replicates and extends [Jha, Qian, Weber & Yang (2024) — "ChatGPT and Corporate Policies"](https://www.nber.org/papers/w32161) (NBER WP 32161) using open-source large-language models to score S&P 500 earnings call transcripts for expected capital investment, then validates those scores against realized Compustat capex and CRSP stock returns.
